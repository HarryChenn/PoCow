# 德牛 3+2 · PoCow

自创卡牌游戏「德牛 3+2」的网页单机版：1 名玩家对战 2–7 个 AI，纯前端，无需服务器。

## 运行

```bash
npm install
npm run dev    # 打开提示的本地地址即可游玩
npm test       # 引擎规则单元测试
npm run build  # 产出 dist/ 静态文件
```

## 玩法规则

- 54 张牌（含 2 张 Joker）。J/Q/K/Joker 均算 10 点，A 算 1 点。Joker 不算花色。
- 每人 5 张牌，3–8 人。

### 换牌阶段（每局一次机会，二选一）

1. **与牌堆换牌**：局开始且尚未行动时，弃掉指定一张、从牌堆摸一张。换后本局**退出与对手的换牌**——自己不能再发起，别人也不能再指定你交换。
2. **找对手换牌**：指定一名对手发起交换，对方可拒绝；接受则**双方各从对方手牌中暗选一张互换**（看不到牌面，按位置选）。每人每局最多发起 2 次；被拒绝不消耗次数，但不能再向同一人发起。

### 牌型与赔率

5 张拆为 3 张**底牌**（定倍数）+ 2 张**踢脚**（定基数），赔率 = 基数 × 倍数。游戏自动选择最优拆分。

**底牌（3 张，点数和为 10 的倍数即成牛）**

| 底牌加成 | 倍数 |
| --- | --- |
| 同花 | 2× |
| 顺子（Q+K+Joker 视为顺子） | 2× |
| 三条 | 3× |
| 王炸（双 Joker） | 3× |

同一组 3 张同时满足多个加成时相乘（如 3 张同花顺 = 4×）。

**踢脚（2 张，点数和取个位）**

| 踢脚 | 基数 |
| --- | --- |
| 1～6 | 1× |
| 7 / 8 / 9 | 2× / 3× / 4× |
| 0（牛牛） | 5× |
| 对子 / 双 Joker | 7× |

**特殊胜利**（整手 5 张，无需凑牛；不叠踢脚）

| 牌型 | 基数 |
| --- | --- |
| 五张顺子 | 8× |
| 五张同花 | 9× |
| 五花（全为人头牌/Joker） | 10× |
| 十小（点数和 ≤ 10） | 11× |
| 炸弹（四条） | 12× |

特殊胜利赔率 = 特殊基数 × 底牌加成倍数：

- 同时满足多个特殊胜利时基数**相加**（同花顺 = 8+9）；比牌力时只取最高基数。
- 底牌加成取 5 张中「加成乘积最大的**某一个** 3 张子集」；加成只在同一子集自身同时满足时相乘，**不能跨子集拼凑**。
  - 五张顺子自带 3 张顺 → 8×2 = 16×
  - 炸弹自带三条 → 12×3 = 36×
  - 同花顺 → (8+9)×2×2 = **68×**

### 比牌与结算

- 牌力只比**特殊胜利基数 / 踢脚基数**，底牌倍数不参与比大小、仅参与结算。
- 无牛（凑不出 10 的倍数底牌）仍参与比较，小于任何有牛的手牌。
- 牌力相同（含全员无牛）时按**德州扑克规则**比 5 张牌（Joker 视为最大单牌）。
- **赢家通吃**：牌力最高者获胜，每个输家按赢家赔率赔分；无牛获胜按 1× 结算；并列获胜时均分。

## 工程结构

```
src/engine/   纯 TypeScript 游戏引擎（与 UI 解耦，全部可单测）
  cards.ts    牌型定义、点数、牌堆
  scoring.ts  底牌倍数、踢脚基数、特殊胜利、最优拆分
  compare.ts  牌力比较 + 德州扑克平局判定
  game.ts     发牌 → 换牌 → 摊牌 → 结算 状态机
  ai.ts       AI 启发式决策
src/ui/       React 组件
```
